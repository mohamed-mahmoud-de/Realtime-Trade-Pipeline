End this session with a structured recap:

1. ARCHITECTURE UPDATE — draw the current ASCII architecture
   showing what components now exist vs what is still unbuilt

2. WHAT WE BUILT — concrete things added or changed today

3. CONCEPTS LEARNED — streaming concepts I practiced today.
   For each concept, write one sentence I could say in an interview.

4. WHAT BROKE — errors we hit, what they meant, how we fixed them

5. STILL INCOMPLETE — what is unfinished from today

6. HOMEWORK — one small thing to try on my own before next session.
   Should be doable in 15-20 minutes without guidance.

7. COMMIT MESSAGE — suggest a clear git commit message

After the recap, remind me to commit, push, and update
the session log in Notion.
